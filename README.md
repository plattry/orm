## What is it
Orm is a simple database abstraction layer that supports postgresql.

_Welcome to __join us__ or __star us__ for encouragement._

## Requires
- php __8.1__ or above
- enable extension __pdo__

## Installation
```shell
$ composer require plattry/orm
```
